## ğŸ‡ºğŸ‡¸ English

This project is a **Study Log Web App built purely with HTML, CSS, and
JavaScript**.\
Its main purpose is to clearly demonstrate **strong foundational web
development skills** without relying on frameworks.

### â— Technical Highlights

-   Semantic HTML structure\
-   Minimal Japanese-style UI design with clean spacing\
-   Responsive layout via media queries\
-   Vanilla JavaScript DOM handling\
-   Fully implemented CRUD (Create / Read / Update / Delete)\
-   Category filtering and keyword search\
-   Persistent data storage using `localStorage`

### â— Purpose of the Project

-   Japanese companies often value **solid fundamentals** more than
    framework skills for entry-level roles\
-   Therefore, this project intentionally avoids frameworks/DBs to
    highlight:
    -   Code clarity\
    -   Understanding of browser behavior\
    -   Ability to build functional UI/UX with raw web technology\
-   The structure can be expanded to future projects using Next.js and
    real databases (e.g., stock/crypto apps)

This Study Log App is intentionally simple---but technically
complete---to highlight core web development ability through clean,

It serves as both a **portfolio piece for job applications** and a
**foundation** for more advanced applications in the future.
